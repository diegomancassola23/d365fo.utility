This repository serves as a **collection of utilities and tools for Microsoft Dynamics 365 Finance & Operations**. Each tool is contained in its own folder with its own README and installation instructions.

---

## 📌 Overview

This repository consolidates various utilities designed to improve development and testing workflows in **D365FO**.  
Each utility is **self-contained**, allowing independent use and deployment.

---

## 🛠 Available Tools

### ZPL Preview Utility

**Description:**  
A lightweight tool to preview ZPL labels directly inside D365FO without requiring local virtual printers.  

> ⚠️ Intended for development and testing environments. Not for production-grade label management.
