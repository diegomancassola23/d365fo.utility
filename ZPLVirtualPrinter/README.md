A lightweight utility that allows previewing ZPL labels directly inside Microsoft Dynamics 365 Finance & Operations without requiring local virtual printer installation.

## üìå Overview

Testing ZPL labels in D365FO often requires installing local virtual printers or using external tools.  
This utility simplifies the process by:

- Capturing the raw ZPL generated by the system
- Rendering a preview using the Labelary API
- Storing label history for debugging and validation

---

## üèó Architecture

High-level flow:

1. A virtual printer is created in D365FO
2. ZPL output is intercepted
3. Raw ZPL is stored in a custom history table
4. ZPL is optionally sent to the Labelary API
5. Rendered image is returned and stored

The Labelary API is used for rendering:
https://labelary.com/service.html
